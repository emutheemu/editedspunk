CSS

An understanding of HTML is crucial to starting CSS, as CSS adds style to your HTML.

Adding CSS can completely control how your web pages look without changing your HTML. For example, you can build a wall with HTML, but you will need CSS to paint it and round off the edges.

CSS allows you to create rules that specify how the content of an element should appear. For example, you can specify that the background of the page is green, all paragraphs should appear in black using the Impact typeface, or that all level one headings should be in blue, bold, Times typeface.

The key to understanding how CSS works is to imagine that there is an invisible box around every HTML element. CSS allows you to create rules that control the way that each individual box and the contents within it, is presented.

For example:

BOXES
-Width and height
-Borders (color, width, and style)
-Background color and images
-Position in the browser window.
TEXT
-Typeface
-Size
-Colour
-Italics, bold, uppercase,
-Lowercase, small-caps.
HOW DOES IT WORK?
CSS works by associating rules with HTML elements. These rules govern how the content of specified elements should be displayed. A CSS rule contains two parts: a SELECTOR and a DECLARATION as seen in the following image:

-SELECTORS indicate how the elements referred to in the selector should be styled.
-DECLARATIONS indicate how the elements referred to in the selectorr should be styled.
-Declarations are also split into two parts, a property and a calue and are seperated by a colon.
-CSS rules usually appear externally in a seperate document saved in the same file as your index.html doc, although they also may appear within the HTML page using inline CSS by placing them inside a < style > element, which usually sits inside the < head > element of the page.

Colour

There are many online tools to get creative with colour for your website. Colour helps to define different sections and to draw the user's eye to certain information. Complimentary colours are important to the overall accessibility and look of the website.

Click on the colour wheel to go to Palleton.com, where you can identify complimentary colours and copy and paste what is known as a Hex code.

The colour property allows you to specify the color of text inside an element. You can specify any color in CSS in one of three ways:

-RGB VALUES
-COLOUR NAMES
-HEX CODES

The Importance of Contrast.

It is importnat to ensure that there is enough contrast between any text and the background colour in order for people to be able to read your website's content. CSS3 has introduced an extra value for RGB colors to indicate opactiy. It is known as RGBA

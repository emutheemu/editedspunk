Javascript

Along with HTML and CSS, JavaScript is one of the three main contributors to making interesting and dynamic web pages. It enables web pages to be interactive and thus is an essential part of web applications. Most websites use it and all major web browsers have a devoted JavaScript engine to execute it. Lets look at how using Javascript can add interactivity to HTML pages.

Javascript can be directly embedded into your HTML page, so no external file is needed. It is object-based code, uses built in extensible objects but no classes or inheritance.

Javascript can be used to react to user or page events. For example, on a shopping website after a 30 second timer, a pop up may appear in order to try and sell you something, such as a subscription to the site. This pop up is written in Javascript, the timer included.

Other examples of how Javascript is used may be to read and write HTML elements, to validate data, to detect the users browser and to create cookies. Declare javascript code in an XHTML document using the < script >< /script > tags
